# Silabs_AWS_IoT_Gateway
Platform to connect SENTIMATE (EFR32MG21+ZGM130) to AWS via Growhouse Gateway (Dragonboard410C+Growhouse Mezzanine Card)

This repository contains following:

- **Home Automation & Environmental Sensing** demo with AWS IoT cloud platform
  - Growhouse Gateway (Dragonboard410C+Growhouse Mezzanine Card)
  - SENTIMATE (EFR32MG21+ZGM130) sensor (Temperature, Humidity, PIR, and CO2 sensing)
  - AWS IoT cloud platform
